# multi-agent-educational-software_V1

## Run frontend
```bash
cd frontend
npm run dev
```

## Run backend
```bash
cd backend
npm run dev
```

## Run tests
```bash
# deterministic commands are defined in TEST.md
docker compose up -d
docker compose ps
```

## Optional docker compose usage
```bash
docker compose up -d
```

## Orchestrator
```bash
python3 orchestrator.py --mode B --verbose
```
